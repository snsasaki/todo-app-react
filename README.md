# Todoアプリ

## 概要

reactの学習用リポジトリ

## 機能一覧

### 必須要件

- Todo追加
- Todo一覧表示
- Todo完了トグル
- Todo削除
- 空文字防止

### 推奨要件

- フィルタリング

## 技術スタック

- JavaScript
- React
- CSS

## アーキテクチャ

```shell
src/
├── components/
│   ├── Header.tsx
│   ├── TodoInput.tsx
│   ├── TodoList.tsx
│   └── TodoItem.tsx
│
├── layouts/
│   └── MainLayout.tsx
│
├── pages/
│   └── Home.tsx
│
├── App.tsx
└── main.tsx
```

## クイックスタート

```shell
$ npm install
$ npm run dev
```
